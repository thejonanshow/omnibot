🚀 DEPLOYMENT PROCESS:

When you run ./deploy.sh:

1. Checks Node.js (installs if needed)
2. Checks Cloudflare login
3. Prompts for API keys:
   - Opens https://console.anthropic.com
   - Opens https://platform.openai.com
   - Opens https://console.groq.com
   - Opens https://makersuite.google.com
4. Generates shared secret (HMAC key)
5. Creates Cloudflare KV namespaces
6. Deploys worker to Cloudflare
7. Deploys frontend to Cloudflare Pages
8. Optional: Deploys to Runloop
9. Returns production URLs

Takes ~5 minutes.
